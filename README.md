Semester Project 1 - A Collaboration 🦕🦕

[Live demo ](https://furuknappen.github.io/semester-project-2025/)

This is a our Semester Project where we decided to work together! It is a website designed and created for a Community Museum for Kids aged 7-15 and their parents. It is an informative and fun site which is easy to navigate. 

Working together on this project allowed us to get used to working on the same source code, and getting more comfortable with branches, git and solving potential issues. 🪼🐱

For this projects design process we decided to create something that would resonate with the age group. The site is fully responsive and accessible! 

Technologies used: HTML and CSS. 💻
